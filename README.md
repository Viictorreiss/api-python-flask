
# API em Python (Flask)

É uma API simples com metodos GET, PUT, POST E DELETE

## Variáveis de Ambiente

Para rodar esse projeto, você vai precisar adicionar as seguintes variáveis de ambiente no seu .env

`API_KEY`

`ANOTHER_API_KEY`


## Instalação

Instale my-project com npm

```bash
  python3 install
  pip install
  pip install flask
```
